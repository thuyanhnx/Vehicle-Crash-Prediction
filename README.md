# Frequency and Severity of Vehicle Crash Prediction using Statistical Models

The project aims to investigate the contributing factors associated with vehicle crashes by focusing on two core research questions: predicting the **frequency of crashes** (enabling proactive identification of high-risk factors), and predicting the **severity of crashes** (optimizing emergency response and safety precautions). This project uses statistical models: linear regression and logistics regression, together with necessary log-transformation and interactive variables. 

The full report paper of the project can be accessed via [Project Report](https://drive.google.com/file/d/1yuYTTiBlH5DWtGc5cufmjtLtJvXlsP2w/view?usp=sharing).
